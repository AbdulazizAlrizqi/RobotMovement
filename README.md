# RobotController
this webpage contain html and php that connected to mysql database to control the robot in 4 directions.

## movement.html File
the html file contains the design of the page and handle the component like the buttons and title in main page

directions:
- Forward
- Backward
- Right
- Left
- and Stop the robot

this file connect with movement.css file

## movement.css File
this file handle the position of each button in the above html file and contain the colors and effeicts in the page

## direction.php file
this file contain the backend of the website it connect the website with the database so when the user click any button the code take this button value and store it in the database
